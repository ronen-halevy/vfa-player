About the demo app

VfbfStreamer is a video streamer utility for frame-by-frame video processing applications

This demo demonstrates usage of VfbfStreamer.

It presents implementation with various data sources

Note: Beside streaming video frame-by-frame, the streamer supports forwarding of still images. Motivation for supporting images by streamer was to supply an integrative contentionless module for apps which run both images and videos on the same canvas
